# ISP Backend - Gestión de Usuarios y Red

## Descripción

Este proyecto es una API REST desarrollada con Spring Boot para la gestión de usuarios dentro de un entorno de red (ISP).
Permite realizar operaciones CRUD, manejar  y aplicar buenas prácticas de desarrollo backend.


## Tecnologías utilizadas

* Java 17+
* Spring Boot
* Spring Data JPA (Hibernate)
* Oracle Database
* Maven
* Lombok
* Postman (para pruebas)


## Arquitectura

El proyecto implementa una arquitectura basada en:

* MVC (Modelo - Vista - Controlador)
* DAO (Data Access Object)
* DTO (Data Transfer Object)

Además, se aplican buenas prácticas como:

* Manejo global de excepciones
* Respuestas estandarizadas (ApiResponse)
* Separación de responsabilidades por capas
